We will have grade wise and subject wise curriculum.
Each subject will have a module (or chapter) and within each module we will have atoms.

For Mathematics (Maths), we will also have potential misconceptions within atom to track which misconceptions the student has.

For rest of the subjects, we will not track misconception.

We will have these subjects:
Subjects covered as part of curriculum (CBSE or Telangana State Board)
-> English (mostly grammar etc)
-> Science (physics, chemistry, biology)
-> Social (history, geography, civics, economics)
-> Mathematics (maths)

Supplemental Subjects:
-> Vocabulary
-> GK (General Knowledge)
-> Reasoning
